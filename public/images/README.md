# Images

Drop image files in here using the **exact filenames** below. The app looks them up by name — anything missing falls back to a generated placeholder, so you can add them a few at a time.

**Format** — `.jpg` or `.webp` preferred (`.png` works). Class art is displayed wide, subclass art is displayed square.

| Folder | Aspect | Suggested size |
|---|---|---|
| `classes/` | 3:2 landscape | 1200 × 800 |
| `subclasses/` | 1:1 square | 800 × 800 |
| `ui/` | anything | backgrounds, textures, logo |

- `stone.jpg` — dark-theme background texture
- `logo.png` — wordmark
